Cheatsheets Interactives

Les commandes et syntaxes essentielles à portée de main.

Navigation fichiers
ls -la          # lister avec détails
cd /home/user   # changer répertoire
pwd             # répertoire courant
mkdir dossier   # créer dossier
rm -rf dossier  # supprimer
cp src dest     # copier
mv src dest     # déplacer/renommer
Processus & Système
ps aux          # voir les processus
kill -9 PID     # tuer un processus
top             # moniteur système
htop            # moniteur amélioré
df -h           # espace disque
free -h         # mémoire RAM
uname -a        # info système
Réseau & Permissions
chmod 755 fic   # droits fichier
chown u:g fic   # propriétaire
ifconfig        # interfaces réseau
ping 8.8.8.8    # tester connexion
ssh u@host      # connexion SSH
scp fic u@host: # copier via SSH
netstat -tulnp  # ports ouverts
Recherche & Fichiers texte
grep -r mot .   # chercher dans fichiers
find / -name f  # trouver fichier
cat fichier     # afficher
less fichier    # paginer
head -n 20 f   # 20 premières lignes
tail -f log    # suivre logs en direct
wc -l fichier  # compter les lignes
Structures de base
# Listes
lst = [1, 2, 3]
lst.append(4)
lst.pop()

# Dictionnaires
d = {'clé': 'val'}
d.get('clé', 'défaut')

# Tuple, Set
t = (1, 2)   s = {1, 2, 3}
Fonctions & Classes
def ma_fonction(a, b=0):
    return a + b

# Lambda
f = lambda x: x**2

class Animal:
    def __init__(self, nom):
        self.nom = nom
    def parler(self):
        pass
Compréhensions & Itérations
# Compréhension liste
carres = [x**2 for x in range(10)]

# Filtrage
pairs = [x for x in lst if x%2==0]

# Enumerate et zip
for i, v in enumerate(lst): ...
for a, b in zip(l1, l2): ...
Fichiers & Exceptions
with open('f.txt', 'r') as f:
    contenu = f.read()

try:
    x = int('abc')
except ValueError as e:
    print(e)
finally:
    print('toujours')
Requêtes de base
SELECT col1, col2
  FROM table
 WHERE condition
 ORDER BY col1 DESC
 LIMIT 10;

SELECT DISTINCT ville FROM clients;
SELECT COUNT(*) FROM commandes;
Jointures
-- INNER JOIN
SELECT * FROM A
INNER JOIN B ON A.id = B.a_id;

-- LEFT JOIN
SELECT * FROM A
LEFT JOIN B ON A.id = B.a_id;

-- Jointure multiple
FROM A JOIN B ON ...
        JOIN C ON ...
Agrégation
SELECT dept,
       COUNT(*) AS nb,
       AVG(salaire),
       MAX(salaire)
  FROM employes
 GROUP BY dept
HAVING COUNT(*) > 5;
DDL & DML
CREATE TABLE users (
  id   INT PRIMARY KEY,
  nom  VARCHAR(100)
);

INSERT INTO users VALUES (1,'Ali');
UPDATE users SET nom='Sara'
 WHERE id=1;
DELETE FROM users WHERE id=1;

git fetch origin
git tag v1.0          # créer un tag

Roadmap étudiant IT

Un parcours progressif de la 1ère année jusqu'à l'insertion professionnelle.

Semestre 1 — Fondations

Algorithmique & Mathématiques Informatiques

Logique, pseudo-code, structures de données de base, algèbre booléenne et bases de la programmation en C.

Semestre 2 — Systèmes & Réseaux

Linux, OS et premiers pas en réseau

Système de fichiers, processus, commandes shell, modèle OSI, TCP/IP et adressage IPv4.

Semestre 3 — Développement

Web, Bases de données & POO

HTML/CSS/JS, SQL, conception BDD (MCD/MLD), programmation orientée objet en Python ou Java.

Semestre 4 — Spécialisation

Réseaux avancés & Sécurité

Routage dynamique, VLAN, VPN, cryptographie, OWASP, pare-feu et premières notions télécom.

Semestre 5 — Cloud & IA

Cloud, Docker & Intelligence Artificielle

Conteneurisation, AWS, machine learning avec Python, Kubernetes et architecture microservices.

Semestre 6 — Projet & Stage

Projet de fin d'études & Insertion pro

Mémoire technique, génie logiciel (UML, Agile, Git), stage en entreprise et préparation certifications (CCNA, AWS, LPIC).
